neodroidagent.utilities.signal.experimental.nstep.discounted_nstep¶

neodroidagent.utilities.signal.experimental.nstep.discounted_nstep(signals: ndarray, values: ndarray, terminals: ndarray, discount_factor, n) → ndarray[source]¶

Implementation of n-step reward given by the equation:

\[G_{t:t+n} = R_{t+1} + \gamma R_{t+2} + \cdots + \gamma^{n-1} R_{t+n} + \gamma^n V_{t+n-1}(S_{t+n})\]

  • values –

  • terminals –

  • discount_factor – discount value. Should be between (0, 1]
